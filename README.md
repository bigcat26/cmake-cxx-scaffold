# CXX Quick Start Project

A skeleton project aim to start my C++ project as quick as possible

## Requirements

- CMake
- Ninja
- VSCode

## Usage

```shell
git clone https://github.com/bigcat26/cxx-starter.git
code -a cxx-starter
```

- `cmd + shift + p:  CMake Configure`
- `cmd + shift + p:  CMake Build`
- `F5`: start debug run
